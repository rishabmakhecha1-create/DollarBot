What To Check

  • Cloudflare Pages deployed the latest GitHub version
  • The `AI` Workers AI binding exists in project settings
  • `/api/health` returns JSON instead of HTML
  • Your browser is hard refreshed after a new deploy

Fallback Behavior

If live AI has a temporary issue, DollarBot keeps working in built-in guidance mode so the beta does not go dark. That means the app can still answer basics and the premium tools still work.
